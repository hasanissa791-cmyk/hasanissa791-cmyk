About

I'm an AI automation engineer working on AI agents and workflow automation — systems that connect LLMs to real tools (APIs, CRMs, spreadsheets, messaging apps) to reduce manual, repetitive work.

I'm currently building my portfolio of AI agent projects and looking for freelance work or a remote position where I can apply and grow these skills on real problems.

What I work with:


AI agents (task planning, tool calling, memory)
Workflow automation (n8n, Zapier, custom scripts)
LLM-powered applications (chatbots, assistants, document processing)
API integrations (CRMs, Slack, Notion, Google Workspace)
